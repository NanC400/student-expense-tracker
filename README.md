# 🎯 Student Expense Tracker  

A mobile/web app to help students track rent, mess, and other expenses easily.  

## 🚀 Features  
✅ Add & track expenses  
✅ Generate digital receipts  
✅ UPI integration  

🔨 *Tech Stack:* HTML, CSS, JavaScript, Python, MySQL
